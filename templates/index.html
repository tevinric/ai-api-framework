Getting Started

Follow these simple steps to start building with our AI API Platform

1

Register for an API Key

Request your unique API key through our registration form to gain access to all services.

2

Generate a Token

Use your API key to generate an authentication token for secure API access.

// Example token generation
POST /token
Headers: {
X-API-Key: your_api_key
}
3

Make Your First API Call

Start with a simple request to one of our language models or other services.

// Example LLM API call
POST /llm/gpt-4o-mini
Headers: {
X-Token: your_token
}
Body: {
"system_prompt": "You are a helpful assistant.",
"user_input": "Tell me about AI APIs."
}
4

Monitor Your Usage

Track your API credit consumption and balance through our monitoring endpoints.
