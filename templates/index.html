Getting Started

Start building with our AI API Platform in just a few steps:

  1. Sign up for an account - Request your API key to access our services.
  2. Generate your first token - Using your API key, generate an authentication token to access the services.
  3. Make your first API call - Start with a simple request to one of our language models or other services.
  4. Monitor your usage - Keep track of your API calls and credit consumption through our balance management endpoints.
