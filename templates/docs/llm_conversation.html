Conversational AI

Build powerful, persistent conversational experiences with GAIA's Conversational API. Create, continue, and manage conversations across multiple LLM models and specialized assistant types.

Download Postman Collection API Documentation

How It Works

1

Create a conversation

Initialize a new conversation with your preferred LLM model and assistant type

2

Get a response

Receive a contextual response from the AI assistant which includes a unique conversation ID for perpetuating the conversation.

3

Continue a chat

Send follow-up messages using the received conversation ID in subsequent messages withing the same conversation.

4

Manage History

Access or delete conversation history as needed

Key Features

  • Multiple LLM Support - Choose from a variety of models including GPT-4o, GPT-4o-mini, O1-mini, DeepSeek-R1, and Llama models (more to come)
  • Persistent Conversations - Conversations are stored securely and can be continued at any time
  • Context Management - GAIA automatically manages context between messages for coherent conversations
  • Token Usage Tracking - Monitor input and output tokens for each conversation
  • Specialized Assistants - Choose from general, coding, creative, research, and business assistant types

Assistant Types

General Assistant

All-purpose AI assistant for a wide range of tasks

  • Answers questions across many domains
  • Helps with daily tasks and problem-solving
  • Provides information and explanations
  • Offers helpful suggestions and ideas
Best for:

General questions, everyday assistance, and diverse information needs

"Can you explain how solar panels work and what factors affect their efficiency?"

Coding Assistant

Specialized in programming and development tasks

  • Writes, explains, and debugs code
  • Supports multiple programming languages
  • Helps with algorithm design
  • Offers best practices and optimization tips
Best for:

Developers, programmers, and anyone working with code

"Write a Python function that checks if a string is a palindrome, with comments explaining the logic."

Creative Assistant

Focused on creative writing and content generation

  • Helps with writing stories, poems, and scripts
  • Generates creative content ideas
  • Assists with brainstorming and ideation
  • Offers feedback on creative work
Best for:

Writers, content creators, marketers, and creative professionals

"Write a short story about a time traveler who accidentally changes history in a small but significant way."

Research Assistant

Specialized in in-depth analysis and information

  • Provides thorough, well-structured information
  • Helps with data analysis and interpretation
  • Explains complex concepts in detail
  • Presents balanced perspectives on topics
Best for:

Researchers, students, analysts, and in-depth learning

"Provide a comprehensive analysis of the environmental impacts of electric vehicles compared to traditional combustion engines."

Business Assistant

Specialized in professional communication and business topics

  • Helps with professional emails and communications
  • Assists with business strategy and planning
  • Provides market and industry insights
  • Offers formal and professional content
Best for:

Business professionals, executives, and corporate communications

"Draft a professional email to a client explaining a project delay and proposing a new timeline."

Create a New Conversation

Start a new conversation with your chosen LLM model and assistant type.

# Create a new conversation
curl -X POST 'https://tihaiapis.tihsa.co.za/llm/conversation/chat' \
  -H 'Content-Type: application/json' \
  -H 'X-Token: YOUR_API_TOKEN' \
  -d '{
    "llm": "gpt-4o",
    "assistant_type": "coding",
    "user_message": "Write a function to calculate the Fibonacci sequence in Python",
    "temperature": 0.7
  }'

Continue an Existing Conversation

Send a follow-up message to an ongoing conversation using the conversation ID.

# Continue an existing conversation
curl -X POST 'https://tihaiapis.tihsa.co.za/llm/conversation/continue' \
  -H 'Content-Type: application/json' \
  -H 'X-Token: YOUR_API_TOKEN' \
  -d '{
    "conversation_id": "a1b2c3d4-5678-90ab-cdef-ghijklmnopqr",
    "user_message": "Can you modify the function to use recursion instead?"
  }'

Delete a Conversation

Remove a conversation and its history when no longer needed.

# Delete a conversation
curl -X DELETE 'https://tihaiapis.tihsa.co.za/llm/conversation?conversation_id=a1b2c3d4-5678-90ab-cdef-ghijklmnopqr' \
  -H 'X-Token: YOUR_API_TOKEN'

Response

{
  "message": "Conversation deleted successfully",
  "conversation_id": "a1b2c3d4-5678-90ab-cdef-ghijklmnopqr"
}
