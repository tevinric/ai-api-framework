Document Intelligence API

Extract valuable insights from your documents with our powerful Document Intelligence APIs. From layout analysis to text extraction and intelligent summarization, transform unstructured documents into structured, actionable information.

Download Postman Collection Swagger API Docs

Overview

Our Document Intelligence APIs provide powerful tools for extracting information, analyzing layout, and summarizing documents. These services help you transform unstructured document data into structured information that can be easily processed and analyzed.



Authentication Required

All Document Intelligence endpoints require authentication using a token. You must include your token in the X-Token header for all requests.

See the Token Services documentation for details on token generation.

Layout Analysis

Extract structured information from documents, including tables, checkboxes, and complex layouts for forms, invoices, and formatted content.

Document Reading

Extract clean, formatted text from documents with high accuracy, preserving paragraph structure and content organization.

Document Summarization

Generate concise, intelligent summaries of lengthy documents with key points extraction and content analysis.

Multi-Format Support

Process a wide range of document formats including PDFs, images, Word documents, PowerPoint presentations, and Excel spreadsheets.

Layout Analysis
5 AI Credits

The Layout Analysis endpoint extracts structured information from documents, including tables, checkboxes, and text layout. This service is useful for forms, invoices, and documents with complex formatting.

Features

  • Table extraction with structure preservation
  • Form field and checkbox detection
  • Key-value pair identification
  • Rich text content with role classification
  • Barcode and language detection
  • Handwritten content identification
  • Selective page processing

API Endpoint

POST /docint/layout

Extract tables, checkboxes, and text from forms and documents by sending a POST request to this endpoint with your file and configuration options.

Required parameters:

  • file_ids: Array of file IDs to process (uploaded via /file endpoint)
  • X-Token: Your API authentication token (header)

Optional parameters:

  • include_barcode: Enable barcode detection (default: false)
  • include_language: Include language identification (default: false)
  • include_keyvalue_pairs: Include key-value pairs extraction (default: false)
  • output_format: Output format for tables (text or markdown, default: text)
  • page_selection: Page selection strategy (all_pages or range, default: all_pages)
  • page_range: Page range to process (e.g., "1,3-6"), only used if page_selection is "range"

Document Reading
5 AI Credits

The Document Reading endpoint focuses on extracting clean, structured text from documents. This service is optimized for text-heavy documents and provides high-quality text extraction with formatting.

Features

  • High-quality text extraction
  • Paragraph structure preservation
  • Handwritten content detection
  • Language identification
  • Selective page processing
  • Barcode detection (optional)
  • Support for PDF and image formats

API Endpoint

POST /docint/read

Extract printed and handwritten text from images and documents by sending a POST request to this endpoint with your file and configuration options.

Required parameters:

  • file_ids: Array of file IDs to process (uploaded via /file endpoint)
  • X-Token: Your API authentication token (header)

Optional parameters:

  • barcode_detection: Enable barcode detection (default: false)
  • language: Include language identification (default: false)
  • page_selection: Page selection strategy (all_pages or range, default: all_pages)
  • page_range: Page range to process (e.g., "1,3-6"), only used if page_selection is "range"

Document Summarization
1 AI Credit

The Document Summarization endpoint analyzes and summarizes document content. It provides key points, document structure, and concise summaries of lengthy documents.

Features

  • Intelligent document summarization
  • Key points extraction
  • Document structure identification
  • Adjustable summary length and style
  • Multiple document format support
  • Customizable creativity level
  • Optional summary file generation

API Endpoint

POST /docint/summarization

Summarize documents using AI by sending a POST request to this endpoint with your file and configuration options.

Required parameters:

  • file_id: ID of the uploaded file to summarize
  • X-Token: Your API authentication token (header)

Optional parameters:

  • length: Desired summary length (short, medium, long, very_long, default: medium)
  • style: Style of the summary (concise, detailed, creative, technical, narrative, bullet_points, default: concise)
  • include_structure: Whether to include document structure in the summary (default: true)
  • temperature: Creativity level (0.0 = deterministic, 1.0 = creative, default: 0.3)
  • include_file_upload: Whether to create and upload a text file with the summary (default: true)
