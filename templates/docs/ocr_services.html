Overview

GAIA's OCR (Optical Character Recognition) services provide specialized document extraction capabilities for documents such as South African ID documents and vehicle license discs (more coming soon). These services extract structured data from document images and return it in a standardized JSON format.



South African ID Processing

Extract key information from smart ID cards, including personal details, ID numbers, and citizenship information.

Vehicle License Disc Analysis

Automatically extract vehicle information including registration numbers, make, model, VIN numbers, and expiry dates from license discs.

Barcode Processing

Extract data directly from document barcodes (when present) for higher accuracy and more reliable information extraction.

Multiple Format Support

Process images (.jpg, .jpeg, .png), PDFs (.pdf), and TIFF files (.tiff, .tif) for maximum flexibility in your document workflows.

Authentication Required

All OCR services require authentication using a token. You must include your token in the X-Token header for all requests.

See the Token Services documentation for details on token generation.

South African ID Documents

Extract structured information from South African ID documents (Smart ID cards) using advanced OCR technology. The service can extract data from the barcode (when present) or directly from the image using Generative AI models.

Features

  • Extraction from both green ID books and smart ID cards
  • Barcode extraction for higher accuracy
  • Fallback to image OCR using machine learning
  • Multiple file format support (JPG, PNG, PDF, TIFF)
  • Automatic document cleanup and processing
  • Secure document handling with automatic deletion

API Endpoint

POST /ocr/sa_id_card

Extract information from South African ID documents by sending a POST request to this endpoint with your file ID.


Required parameters:

  • file_id: ID of the previously uploaded ID document image
  • X-Token: Your API authentication token (header)

File upload process:

  1. First, upload your ID document image using the /file endpoint
  2. Receive a file_id for your uploaded image
  3. Submit this file_id to the OCR endpoint
  4. Receive structured ID information in response

Example


Request Details

API Endpoint
POST /ocr/sa_id_card
Headers
X-Token: your-authentication-token
Request Body
{ "file_id": "your-file-id" }
Supported File Types
Images (.jpg, .jpeg, .png), PDF (.pdf), TIFF (.tiff, .tif)

Pro Tips for ID Document OCR

  • For best results, ensure the ID document is well-lit and the entire document is visible in the image
  • Position the document against a dark background for better contrast
  • Barcode extraction (when available) is more accurate than image OCR
  • For Smart ID cards, scan both sides but submit the side with the ID number and personal details

Vehicle License Disc


Extract detailed information from South African vehicle license discs using OCR technology. The service can extract data from the barcode (when present) or directly from the license disc image using machine learning.

Features

  • High-fidelity extraction from vehicle license discs
  • PDF417 barcode extraction for higher accuracy
  • Fallback to image OCR using machine learning
  • Multiple file format support (JPG, PNG, PDF, TIFF)
  • Extraction method indication in response
  • Secure document handling with automatic deletion

API Endpoint

POST /ocr/vehicle_license_disc

Extract information from South African vehicle license discs by sending a POST request to this endpoint with your file ID.


Required parameters:

  • file_id: ID of the previously uploaded license disc image
  • X-Token: Your API authentication token (header)

File upload process:

  1. First, upload your license disc image using the /file endpoint
  2. Receive a file_id for your uploaded image
  3. Submit this file_id to the OCR endpoint
  4. Receive structured vehicle license information in response

Example

Request Details

API Endpoint
POST /ocr/vehicle_license_disc
Headers
X-Token: your-authentication-token
Request Body
{ "file_id": "your-file-id" }
Supported File Types
Images (.jpg, .jpeg, .png), PDF (.pdf), TIFF (.tiff, .tif)

Pro Tips for License Disc OCR

  • For best results, capture the entire license disc including the barcode
  • Ensure the barcode (PDF417) is clearly visible for more accurate extraction
  • Remove the license disc from the windshield to avoid glare and reflections
  • Position the disc against a dark background for better contrast
  • When extraction_method is "barcode analysis", the data is extracted directly from the PDF417 barcode and is highly accurate
