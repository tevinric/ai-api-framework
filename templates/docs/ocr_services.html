OCR Services API

South African ID Document OCR

Extract information from South African ID documents using OCR technology.

Endpoint

POST /ocr/sa_id

Headers

  • X-Token: Your authentication token (required)
  • Content-Type: application/json

Request Body

{
    "file_id": "your-file-id"
}

The file_id parameter should be an ID of a previously uploaded file using the /file endpoint.

Supported File Types

The following file formats are supported:

  • PNG (.png)
  • JPEG (.jpg, .jpeg)
  • PDF (.pdf) - Single page only
  • TIFF (.tiff, .tif)

Response

Success (200 OK):

{
    "surname": "DOE",
    "names": "JOHN",
    "sex": "M",
    "nationality": "RSA",
    "identity_number": "0123456789012",
    "date_of_birth": "01 JAN 1990",
    "country_of_birth": "SOUTH AFRICA"
}

Error Responses

  • 400 Bad Request: Invalid parameters or unsupported file type
  • 401 Unauthorized: Missing or invalid token
  • 500 Server Error: Server processing error

CURL Example

curl -X POST \
  'https://your-api-domain.com/ocr/sa_id' \
  -H 'X-Token: your-token-here' \
  -H 'Content-Type: application/json' \
  -d '{
    "file_id": "12345678-1234-5678-1234-567812345678"
}'
