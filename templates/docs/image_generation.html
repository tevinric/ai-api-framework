Image Generation

Our API platform provides access to cutting-edge image generation models that can create high-quality images from text descriptions. These models are perfect for creative content generation, design exploration, and visual asset creation.

Download Postman Collection

Authentication Required

All image generation endpoints require authentication using a token. You must include your token in the X-Token header for all requests.

See the Token Services documentation for details on token generation.

Available Models

DALL-E 3

3 credits

OpenAI's most advanced image generation model, capable of creating detailed and accurate images from text prompts.

  • Exceptional text-to-image fidelity
  • Advanced understanding of complex prompts
  • Multiple size options including widescreen
POST /image-generation/dalle3

Stable Diffusion Ultra

2 credits

A powerful open-source image generation model with excellent control and customization options.

  • Fine-grained control with negative prompts
  • Deterministic generation with seed control
  • Versatile style capabilities
POST /image-generation/stable-diffusion-ultra

Image Generation Parameters

Our image generation endpoints support various parameters to customize your generated images:

DALL-E 3 Parameters

  • prompt - Text description of the image to generate (required)
  • deployment - Model deployment to use: dall-e-3 (default)
  • size - Output image size: 1024x1024 (default), 1792x1024, or 1024x1792
  • quality - Image quality: standard (default) or hd
  • style - Image generation style: vivid (default) or natural

Stable Diffusion Ultra Parameters

  • prompt - Text description of the image to generate (required)
  • negative_prompt - Text describing what to avoid in the generated image
  • size - Output image size: 1024x1024 (default), 768x768, or 512x512
  • output_format - Output format: png (default) or jpeg
  • seed - Seed for reproducible generation, -1 (default) for random seed

Model Selection Guide

When choosing which model to use, consider the following factors:

  • Image quality requirements - For maximum quality, use Stable Diffusion Ultra
  • Creative control - For more control over generation, use Stable Diffusion Ultra
  • Photorealism - For photo realistic images of humans, use Stable Diffusion Ultra
  • Budget considerations - Dalle3 offers a good balance of quality and cost

Pro Tip

When crafting prompts for image generation, be specific and detailed. Include information about:

  • Subject matter (what you want to see)
  • Style (photorealistic, cartoon, painting, etc.)
  • Composition (close-up, wide shot, perspective)
  • Lighting and color palette

Example: "A photorealistic close-up portrait of a silver tabby cat with green eyes in a sunlit garden, soft focus background with colorful flowers, professional photography"
