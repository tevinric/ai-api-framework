DALL-E 3
5 AI Credits

OpenAI's advanced image generation model capable of creating detailed, high-quality images from text descriptions with remarkable accuracy.

Features

  • Photorealistic image generation
  • Detailed artistic renderings
  • Excellent understanding of complex prompts
  • Multiple size options (1024x1024, 1792x1024, 1024x1792)
  • Quality control (standard or HD)
  • Style adjustment (vivid or natural)

API Endpoint

POST /image-generation/dalle3

Generate images with DALL-E 3 by sending a POST request to this endpoint with your prompt and configuration options.

Required parameters:

  • prompt: Text description of the image to generate
  • X-Token: Your API authentication token (header)

Optional parameters:

  • deployment: Model deployment (dall-e-3 or dalle3-hd)
  • size: Image dimensions
  • quality: Image quality (standard or hd)
  • style: Image style (vivid or natural)

STABLE DIUFFION - ULTRA
5 AI Credits

A powerful diffusion-based image generation model capable of creating highly detailed images with exceptional quality and creative flexibility.

Features

  • High-fidelity image generation
  • Negative prompt support for fine-tuned control
  • Multiple size options (1024x1024, 768x768, 512x512)
  • Format selection (PNG or JPEG)
  • Seed control for reproducible results
  • Fast generation times

API Endpoint

POST /image-generation/stable-diffusion-ultra

Generate images with Stable Diffusion Ultra by sending a POST request to this endpoint with your prompt and configuration options.

Required parameters:

  • prompt: Text description of the image to generate
  • X-Token: Your API authentication token (header)

Optional parameters:

  • negative_prompt: Text describing what to avoid in the image
  • size: Image dimensions
  • output_format: Image format (png or jpeg)
  • seed: Numerical seed for reproducible generation
