Balance Management

The Balance Management endpoints allow you to check your current API credit balance and monitor usage statistics. These services help you track your API consumption across different endpoints and time periods.

Credit System

Our API platform uses a credit-based system:

  • Each API endpoint consumes a specific number of credits per call
  • Credits are allocated monthly based on your account tier
  • Usage statistics help you monitor and optimize your credit consumption

If you need additional credits, please contact our support team.

Available Endpoints

GET /usage/balance Check your current API call balance
GET /usage/user Get usage statistics for a specific user
GET /usage/department Get usage statistics for a department
PUT /admin/balance Update a user's API call balance (Admin only)

Checking Your Balance

You can check your current API credit balance using the /usage/balance endpoint. This provides information about your current balance, total allocated credits, and usage statistics.

# Example response from /usage/balance { "user_id": "12345678-1234-5678-1234-567812345678", "user_email": "user@example.com", "current_balance": 2500, "total_allocated": 3000, "credits_used": 500, "reset_date": "2025-04-01T00:00:00Z" }

Monitoring Usage Statistics

The usage statistics endpoints provide detailed information about API consumption for specific users or departments. You can filter statistics by time period to track usage trends.

# Example request to /usage/user { "user_id": "12345678-1234-5678-1234-567812345678", "time_period": "2025-03" } # Example response { "name": "John Doe", "user_id": "12345678-1234-5678-1234-567812345678", "company": "Acme Inc", "department": "Engineering", "tokens_generated": 42, "endpoints_used": [ { "endpoint_name": "GPT-4o", "endpoint_id": "gpt-4o", "number_calls": 20, "number_successful_calls": 19, "number_failed_calls": 1, "average_response_time": 1250.5, "credits_consumed": 40 }, { "endpoint_name": "DALL-E 3", "endpoint_id": "dall-e-3", "number_calls": 5, "number_successful_calls": 5, "number_failed_calls": 0, "average_response_time": 2500.3, "credits_consumed": 15 } ], "credits_consumed": 55 }

Department Statistics

If you're a department manager or administrator, you can view usage statistics for your entire department, including a breakdown by user.

# Example request to /usage/department { "department": "Engineering", "time_period": "2025-03" }

Admin Functions

Administrators can update user balances using the /admin/balance endpoint. This requires admin privileges and a valid token.

# Example request to /admin/balance (admin only) { "user_id": "12345678-1234-5678-1234-567812345678", "new_balance": 5000 }
