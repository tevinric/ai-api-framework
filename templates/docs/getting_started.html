Getting Started with the AI API Platform

This guide will walk you through the steps to start using our AI API Platform, from account setup to making your first API call.

Step 1: Request an API Key

To use the API platform, you'll need an API key with appropriate permissions. Fill out the form below to request your API key:

API Key Request Form

MS Form for API key signup will be embedded here.

Account Types

We offer different account types with varying credit allocations:

  • Developer - For development and testing
  • Professional - For production applications with moderate usage
  • Enterprise - For high-volume production applications

Specify your requirements in the form above, and our team will set up your account accordingly.

Step 2: Download Postman Collection

The easiest way to get started with our API is to use our Postman collection, which includes all endpoints with examples:

Download Postman Collection

Step 3: Generate Your First Token

Before making API calls, you need to generate an authentication token using your API key:

  • Use the /token endpoint with your API key in the header
  • Store the returned token securely
  • Use this token in the X-Token header for subsequent API calls

See our Token Services documentation for more details.

Step 4: Explore Available Endpoints

Our platform offers a wide range of AI capabilities through different endpoints:

  • Language Models - Text generation and analysis
  • Image Generation - Create images from text descriptions
  • Speech Services - Convert speech to text
  • Document Intelligence - Extract insights from documents
  • File Management - Upload and manage files

Step 5: Monitor Your Usage

Keep track of your API consumption using our Balance Management endpoints.

Pro Tip

When developing your application, start with the most cost-effective endpoints (such as GPT-4o-mini) for testing, and switch to more advanced models for production use when needed.
