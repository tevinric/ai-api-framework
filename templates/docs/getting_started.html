Getting Started Process

Follow these steps to start using our AI API Platform

1

Request an API Key

Complete our registration form to request an API key with appropriate permissions for your needs. You can request a key for yourself as an employee or for a service account.

Request Key
2

Download Postman Collection

The easiest way to get started with our API is to use our Postman collection, which includes all endpoints. You can call our API endpoints using any programming language through the collection.

You can also view and test our endpoints directly on the API Swagger documentation.

Download Collection
3

Generate Authentication Token

Once you receive your API key, use it to generate a temporary authentication token. This token will be used for all subsequent API calls to authenticate your requests.

Token Generation POST /token
// Use your API key in header
Headers: {
"API-Key": "your_api_key_here"
}
// Example response
{
"access_token": "eyJhbGciOiJIUzI1NiIsInR5...",
"expires_in": 3599
}
4

Explore Available Endpoints

Browse our comprehensive documentation to discover all available AI services. From language models to image generation, speech services, and document intelligence – choose the endpoints that meet your application's needs.

Swagger API Docs
5

Make Your First API Call

Use your authentication token to make your first API call. Include the token in the header of your request to access any of our AI services. Our consistent API structure makes integration straightforward.

Example API Call POST /llm/gpt-4o-mini
// Include token in header
Headers: {
"X-Token": "your_token_here",
"Content-Type": "application/json"
}

Body: {
"user_input": "Tell me about AI APIs"
}
6

Monitor Your Usage

Keep track of your API consumption using our Balance Management endpoints. Monitor your credit usage across different endpoints and time periods to optimize your AI utilization.

Check Balance GET /usage/balance
Headers: {
"X-Token": "your_token_here"
}

Using the Platform

Essential information to help you start developing with our AI APIs

Token Generation and Authentication

Before making API calls, you need to generate an authentication token using your API key:

  • Use the /token endpoint with your API key in the header
  • Store the returned token securely
  • Use this token in the X-Token header for subsequent API calls
  • Each token is valid for 1 hour (3599 seconds)
  • When using the Postman collection, add the token to the collection environment variables
Token Generation POST /token
Headers: {
"API-Key": "your_api_key_here"
}
// Example response
{
"access_token": "eyJhbGciOiJIUzI1NiIsInR5...",
"expires_in": 3599
}

See our Token Services documentation for more details.
