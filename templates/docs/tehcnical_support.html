Submit a Technical Support Request

Need help with our AI API Platform? Follow these simple steps to submit a technical support request and get assistance from our team.

How to Request Technical Support

Our support team is ready to help you resolve any issues or answer questions about the AI API Platform

1

Sign in to Your Account

Navigate to the AI API Platform portal and sign in with your credentials.

  • Go to https://api-portal.tih.co.za
  • Enter your username and password
  • Click "Sign In"

Sign-in page screenshot

2

Navigate to the Support Section

Once logged in, go to the Support section in the dashboard.

  • Click on your profile icon in the top-right corner
  • Select "Support" from the dropdown menu
  • Alternatively, click on the "Support" link in the left sidebar

Support section navigation screenshot

3

Create a New Support Ticket

In the Support section, create a new support ticket to submit your request.

  • Click the "New Support Request" or "Create Ticket" button
  • Select the appropriate category for your issue from the dropdown menu
  • Common categories include:
    • API Authentication
    • Credits and Billing
    • Technical Issues
    • Feature Requests

Create ticket screenshot

4

Fill Out the Support Form

Complete all required fields in the support form with detailed information about your issue.

  • Enter a descriptive subject line
  • Provide a detailed description of your issue or question
  • Include relevant details such as:
    • Error messages (copy and paste exact text)
    • Steps to reproduce the issue
    • API endpoints you're trying to use
    • Your account ID or username
  • Set the priority level appropriately

Pro Tip

The more information you provide, the faster our team can assist you. If applicable, include error codes, timestamps, and any other relevant technical details.

Support form screenshot

5

Attach Supporting Files (Optional)

If you have screenshots, logs, or other files that help illustrate your issue, attach them to your ticket.

  • Click the "Attach Files" or "Upload" button
  • Select the files from your computer
  • Supported file types include:
    • Images (PNG, JPG, GIF)
    • Text files (TXT, LOG)
    • Documents (PDF, DOCX)
    • Code files (JSON, XML, etc.)
  • Maximum file size: 10MB per file

File attachment screenshot

6

Submit Your Support Request

Review your information and submit your support request.

  • Double-check all provided information for accuracy
  • Click the "Submit" or "Send" button
  • You'll receive a confirmation email with your ticket number
  • Keep this ticket number for reference in follow-up communications

Response Time

Our team typically responds to support requests within 24 business hours. For urgent production issues, we aim to respond within 4 hours during business hours.
