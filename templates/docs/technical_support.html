Submit a Technical Support Request

Need help with our AI API Platform? Follow these simple steps to submit a SysAid request and get assistance from the AI Center of Excellence Team.

How to Request Technical Support

If you encounter any technical difficulties using the AI API platform, you can submit a SysAid request using the procedure below:

1

Open a web browser session

Open up your web browserso that you can acceess the SysAid portal

  • Search for valid web broswer application (Microsoft Edge) in your Window's Search bar
  • Type in "Microsoft Edge"
  • Click on the Mircroft Edge icon to open a browser session.
2

Navigate to the SysAid portal

Enter the web address for accessing the SysAid portal

  • Access SysAid using the following link: https://sysaidsvr1.isservices.co.za/servicePortal
3

Submit a new Request

once on the SysAid Portal, select the option to Submit a Request

  • A new request must be submitted to the AI COE to get assistance.
4

Select the appropiate categories

You must select the correct category to ensure that your request gets directed to the AI COE

  • Select the "Basic Template"
  • Select the category "AI Center of Excellence"
  • Select the sub-category "AI API Portal"
  • Select the third level category related to your reuqest
5

Complete the template fields

Fill out the fields on selected template

  • Specify the details of your request (provide as much information as possible)
  • Different templates may required varying information. Ensure that you provide all information as per the selected template
6

Submit your request

Submit the details of your request to the AI COE.

  • Ensure all required fields are completed
  • Click Submit
  • You'll receive a confirmation email with your SySAid ticket number

Response Time

Please allow 24-48 hours for the AI COE team to review your request and get back to you. The total turn around time will depend on the complexity of the request.
